## Deployment Notes

This project is deployed using:

* Frontend: Vercel
* Backend: Render
* Database: MongoDB Atlas

### Important Note

The backend is hosted on Render's free tier. If the application has been inactive for some time, the backend service may go to sleep.

As a result:

* The first request may take 30–60 seconds to respond while the server wakes up.
* Subsequent requests will work normally once the service is active.

This is an expected behavior of the free hosting plan and does not indicate an issue with the application.
